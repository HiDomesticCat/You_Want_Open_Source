 
## 第一章：我所了解的開源軟體與理念

### 在沒事做時，我特別喜歡瀏覽維基百科，因為時常可以觀看到神人對於某件人事物一針見血的見解，所以我們就以維基上對於[開源軟體的介紹](https://en.wikipedia.org/wiki/Open-source_software)，節錄我覺得重要的片段，來當此章節也是此文章的開始。

> Open-source software (OSS) is computer software that is released under a license in which the copyright holder grants users the rights to use, study, change, and distribute the software and its source code to anyone and for any purpose.Open-source software may be developed in a collaborative public manner. Open-source software is a prominent example of open collaboration, meaning any capable user is able to participate online in development, making the number of possible contributors indefinite. The ability to examine the code facilitates public trust in the software.

>Scholars Casson and Ryan have pointed out several policy-based reasons for adoption of open source – in particular, the heightened value proposition from open source (when compared to most proprietary formats) in the following categories:
>  + Security
>  + Affordability
>  + Transparency
>  + Perpetuity
>  + Interoperability
>  + Flexibility
>  + Localization – particularly in the context of local governments (who make software decisions). Casson and Ryan argue that "governments have an inherent responsibility and fiduciary duty to taxpayers" which includes the careful analysis of these factors when deciding to purchase proprietary software or implement an open-source option.

> 開源軟體是根據許可發布的軟體，其中版權持有者給予用戶「使用」、「學習」、「更改」和「向任何人以任何目的分發軟體及其源代碼」的權力。開源軟體可以用開放式協作的方式進行開發。開源軟體是開放式協作最具代表性的例子，這意味著任何有能力的使用者都可以在線參與開發，從而使可能的參與者數量不定。檢視代碼的能力促進了大眾對於軟體的信任。

> 學者 Casson 和 Ryan 指出了採用開源的幾個策略性因素，特別是以下幾類更是提高了開源的重要性（與大多數專有模式相比下）：
> + 安全性
> + 負擔能力
> + 透明性
> + 永久性
> + 協作性
> + 靈活性
> + 本地化 - 特別是對於地方政府在軟體的選擇上。Casson 和 Ryan 認為「政府有責任與義務對公民負責」，其中就包括決定購買專有軟體還是使用開源軟體時，對於其中因素的仔細分析。

### 從維基百科的介紹中可以找到一個我覺得開源軟體最關鍵也是我最喜歡的地方。

> *給予用戶「使用」、「學習」、「更改」和「向任何人以任何目的分發軟體及其源代碼」的權力*

### 在現今，我們能如此便捷且快樂的生活著，無疑是因為先人無私的付出，在以前當大家有新的發現或創作時，都會興奮地將其給發表出來，不管這樣是否或影響到自己的利益，但產品專有化的出現，打破了原本美好的平衡，大家都將自己的成果隱藏起來，只露出簡單的外表，而封閉了它內在蘊含的技術與價值，保護自己的利益本身是沒有錯的，但當保障自身利益而導致科技發展速度下降，這我覺得就是不對的，可能大家會覺得科技發展的緩慢不一定是專有制度所導致的，那麼我就舉幾個假設來告訴大家文明的發展是需要靠大家無私的貢獻換來的：如果李爺爺（Tim Berners-Lee）將全球資訊網專有化，我們將無法和朋友如此方便的聊天，也無法使遠在各地的我們進行學術交流、如果 linus 大神（Linus Benedict Torvalds）將 linux 的原始碼閉源，我們將沒有辦法如此簡單的創建自己的網站，而且我也沒有電腦來創作此文章、最後如果卡馬克英雄（John D.
